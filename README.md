# Vue PWA Starter

A simple boilerplate to start building progressive web apps with vue js.  Pre-loaded with babel, vue-loader, vue-material, vue-router and sw-precache.

## Getting Started

Install
```
yarn
```

Build for dev
```
yarn dev
```

Build for dev and watch changes
```
yarn watch
```

Hot reloading
```
yarn hot
```

Build for prod
```
yarn prod
```

Run simple http server
```
yarn simple-server
```
